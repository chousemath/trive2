### The Trive ReasonML Library

# Build
```bash
$ npm run build
```

# Build + Watch

```bash
$ npm run start
```

# Compile

```bash
$ npm run-script compile
```

# Test

```bash
$ npm test
```

# Generate Docs

```bash
$ npm run-script docs
```
